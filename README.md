# Packing Planner

Packing Planner is a Python based cli tool for building a packing checklist.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packing_planner.

```bash
pip install packing_planner
```

## Usage

```bash
packplan inventory

packplan plan

packplan trip

```
